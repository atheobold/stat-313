Boxplot

penguins %>%

Density Plot

penguins %>% 
  ggplot(aes(x = bill_length_mm)) + 
  geom_density()

Adding Groups

Side-by-Side Violins

penguins %>% 
  ggplot(aes(x = bill_length_mm, y = species)) + 
  geom_violin(aes(fill = species)) + 
  theme(legend.position = "none")

Faceted Histograms

penguins %>% 
  ggplot(aes(x = bill_length_mm)) + 
  geom_histogram() + 
  facet_wrap(~species)
