Week Eight: Statistical Inference with Hypothesis Tests

Welcome!

In this week’s coursework we are putting all of the pieces we’ve learned together: linear regression, confidence intervals, and p-values! We are going to use your knowledge of linear regression to make inferences about the value of the population slope.

We are going to use permutation distributions to obtain other slope statistics that we could have observed if the null hypothesis was true. We are going to use bootstrap distributions to obtain other slope statistics we could have observed from repeated samples from the population. Then, we will use these distributions to make inferences about the value of the population slope (\(\beta\)).
